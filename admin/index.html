    🔗 GitHub Sync

    Enter your GitHub Personal Access Token to publish posts directly to the live site — no download or upload steps needed. When you click Publish, the post goes live in ~60 seconds.

    How to create a token:
    1. Go to github.com → click your profile → Settings
    2. Developer settings → Personal access tokens → Fine-grained tokens
    3. Generate new token → set repo to atbethesda-website
    4. Under Permissions → Repository permissions → set Contents: Read and write
    5. Generate token → paste it above

    📦 Sync All Posts

    Downloads the full content of all your legacy posts from the live site and stores them locally. After syncing, every post opens instantly with its full content ready — no loading delay, no network call needed.

    ⚠ Make sure your GitHub token is saved above before syncing. This may take a few minutes for 127 posts.
